# C Actions Lab - Basic Arithmetic

이 프로젝트는 두 정수의 합을 계산하고 그 결과를 출력하는 간단한 C 언어 기초 예제입니다.

## 🚀 주요 기능
* **변수 선언 및 초기화**: 두 개의 정수(`int`) 변수 `a`와 `b`를 선언하고 각각 10과 20으로 초기화합니다.
* **산술 연산**: 두 수의 합을 계산하여 `sum` 변수에 저장합니다.
* **표준 출력**: `printf` 함수를 사용하여 변수의 값과 계산 결과를 터미널에 출력합니다.

## 💻 실행 방법

### 1. 컴파일 (GCC 기준)
터미널에서 아래 명령어를 입력하여 코드를 컴파일합니다.

gcc src/main.c -o main
